Detailed Description

Declares the HeadsUpDisplay class for rendering and displaying information on SSD1306/SH1106 displays.

This file defines the HeadsUpDisplay class, which extends the Effect interface to capture and process rendered frames efficiently. It supports SSD1306 displays and provides an untested option for SH1106 displays.
